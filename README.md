# SyncDash

# How to run

0. Create file `.env`
1. Run `make r-env`
2. Run `make build`
3. Run `make run-dev`
4. Run `make create_db`

# How to linter

- If you want to run just linter, run `make linter`

# How to run migrations

1. alembic revision --autogenerate -m `<nome_migration>`
2. alembic upgrade head


- If you want to run just linter, run `make linter`
